What is a 'Botnet infection'?

Your server has been compromised. It's hosting malware and is participating in a botnet.

Why would this be bad?

A botnet can be used to steal your personal data, send spam, hack into other computers and launch network attacks.

Recommended action

This issue needs to be resolved by removing the malicious software.

Tips to resolve this matter

If you suspect a user account has been compromised, change the password for the account immediately.

Scan all hosted websites and temporary directories for suspicious files and remove them. Also check your process list for suspicious processes and kill them.

Install a rootkit and a virus scanner to ensure all malicious files have been removed.

Once all malicious software has been removed make sure the server can't be compromised again by installing the latest updates for your operating system, control panel and hosted applications, including themes and plugins (i.e. Wordpress).
